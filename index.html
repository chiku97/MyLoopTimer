Windows & OS Notification Guide

If you aren't receiving notifications from the OS, ensure the following settings are configured on your system:

  • Browser Permissions: Click the lock/settings icon next to the URL in the address bar and verify that Notifications are set to Allow.
  • Windows Focus Assist / Focus Settings: Search for "Focus Settings" or "Do Not Disturb" in Windows Search. If active, Windows silences all browser/app banners. Make sure it is toggled Off.
  • Windows Notification Center: Navigate to Settings > System > Notifications. Verify that notifications are enabled globally, and that your browser (e.g., Chrome, Edge) is allowed to send banner/sound notifications.
  • Standalone Installation (Highly Recommended): Click Install App in the top header. Running it as an installed Windows app lets the OS treat it like a native calendar or mail client, improving notification reliability.
